## 🎯 Quick Reference

### Cài đặt
```bash
python setup.py
```

### Chạy
```bash
python main.py
```

### Test
```bash
python test_modules.py
```

### Config
```yaml
# Sửa config.yaml
display_width: 1280
camera_width: 640
min_detection_confidence: 0.5
```

### Thêm nội dung
```bash
# Sửa data/pages.json hoặc dùng code
from data_manager import DataManager
dm = DataManager()
dm.add_page(Page(title="...", content="..."))
```
